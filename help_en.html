Chromeculator Help

This calculator uses the powerful math.js library, which allows for a wide variety of calculations.

Unit Conversions

You can convert and calculate with various units on the fly.

  • 5 cm in inch
  • 2 miles to km
  • (100 ft + 5 mph * 10 minutes) in yards

Currency Converter

The calculator uses live exchange rates to convert major world currencies. Use official 3-letter currency codes.

  • Syntax: [amount] [CURRENCY_CODE_1] in [CURRENCY_CODE_2]
  • Example: 15 USD in EUR or 720 UAH to RUB

Financial Calculations (Mixed Currencies)

You can perform calculations by mixing different currencies and plain numbers in a single expression.

  • The calculator will automatically use the first currency found as the base currency for the result.
  • All other currencies will be converted to the base currency.
  • Numbers without a currency are treated as being in the base currency.
  • Example: 100 USD - 25//lunch// + 1500 RUB. The result will be in USD.

Working with Percentages

You can use the % sign to calculate percentages just like on a standard calculator.

  • Add a percentage: 100 + 10% (result: 110)
  • Subtract a percentage: 75 - 25% (result: 56.25)
  • Find a percentage of a number: 50 * 20% (result: 10)

Comments in Calculations

You can leave text notes right inside your calculations. This is useful for budgeting or complex formulas.

  • Syntax: //your comment//
  • Text inside the double slashes will be ignored during calculation but saved in your history.
  • Example: 3600//rent// + 1800//utilities//

Quick History Access

  • Use the Up Arrow and Down Arrow keys in the input field to navigate through your command history.
  • Click on past queries or results to paste them into the input field.

Chained Calculations

You can easily chain calculations by using the result of the previous operation.

  • Auto-substitution: If you start your input with an operator like +, -, *, /, or ^, it will be applied to the last result.
    Example: First, enter 5 * 10 (you get 50), then just type / 2, and the calculator will compute 50 / 2.
  • The ans variable: For more clarity and complex cases, you can use the ans variable, which always holds the last successful answer.
    Example: ans ^ 2 - 10

Function Plotting

You can plot graphs of single-variable functions using the plot command.

  • Syntax: plot(function, [min_x, max_x])
  • Example: plot(x^2 - 2*x + 1, [-5, 5])
  • You can also use constants: plot(sin(x), [-pi, pi])

Custom Functions and Constants

In the Options page, you can define your own functions and constants for frequently used formulas.

  • Constant Example: g = 9.81
  • Function Example: area(r) = pi * r^2
  • Usage: 10 * g or area(10)

Advanced Usage

The math.js library supports complex numbers, matrices, statistics, and much more. For a full list of features, visit the official math.js documentation.
