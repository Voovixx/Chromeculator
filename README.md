## How to Use / –ö–∞–∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å

-   **Plotting:** `plot(sin(x), [-pi, pi])`

-   **Mixed Currency Math:** `100 EUR - 20 + 500 RUB`

-   **Percentages:** `15000 - 13%`
-   **Create a variable:** `myVar = 15 * 4`
