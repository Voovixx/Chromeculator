üöÄ How to Install the Chromecalc Extension

You will need the `Chromecalc` folder with all the extension files.

  1. Open Chrome, then click on the address below to copy it. Paste it into the address bar and press Enter.
    chrome://extensions
  2. In the top right corner, enable Developer mode.
  3. Click the Load unpacked button that appears.
  4. Select the entire project folder (`Chromecalc`).
    You do not need to go inside the folder, just select it and click "Select Folder".
  5. ‚úÖ Done! The calculator icon will appear on your Chrome toolbar.
